# JamPad

> Your game jam launchpad — currently in beta.

Turn any jam theme into a scoped, polished game idea — with proven mechanics and real examples to guide you. Works with Ludum Dare, GMTK, Global Game Jam, and more.

## What it does

- **Theme → Ideas** — Enter your jam theme and get instant game concepts matched to proven mechanics.
- **Scope It Down** — AI-powered scoping ensures your idea fits within your jam's time limit.
- **Real Examples** — Browse real jam winners for inspiration.
- **Mechanic Library** — Explore a curated library of game mechanics tagged by complexity and genre.
- **Time Planner** — Break your jam into phases with realistic milestones.
- **Polish Checklist** — Get a checklist of polish items to elevate your submission.

## How it works

1. **Enter your theme** — Paste the jam theme or constraints you're working with.
2. **Browse ideas** — Get matched game concepts with mechanics, scope, and examples.
3. **Plan & build** — Use the time planner and polish checklist to ship something great.

## Tech stack

React, TypeScript, Vite, Tailwind CSS, shadcn/ui

## Development

```sh
npm install
npm run dev
```
